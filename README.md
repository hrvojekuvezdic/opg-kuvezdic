# Web projekt iz kolegija Osnove razvoja web i mobilnih aplikacija

### Ime projekta: OPG Kuveždić

### Problem koji rješava web aplikacija: Promoviranje obiteljskog poljoprivrednog obrta

### Glavne funkcionalnosti web stranice:
- pregled obrta (vlasnik, lokacija i sl.) i ponude (proizvodi, fotografije, cijene, način uzgoja, dostupnost i sl.),
- detalji o plodovima (fotografije, nutritivna vrijednost, uporaba, priprema, jela i sl.),
- informacije za kupce - kako naručiti, dostava, plaćanje,
- dodavanje proizvoda u košaricu,
- kontakt za slanje upita,
- jednostavna FAQ sekcija s najčešćim pitanjima kupaca.

### Mogućnosti weba koje će biti potrebne:
- HTML,
- CSS,
- JavaScript,
- Google Maps Embed.
